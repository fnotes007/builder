# Offline Website Builder Pro

A powerful, feature-rich drag-and-drop website builder that runs entirely in your browser. No backend required, no plugins needed.

## Features

✨ **30+ Pre-built Sections**
- Hero, About, Services, Pricing, Team, Testimonials
- Blog, Gallery, Slider, Video, Statistics
- Collections, Forms, Custom Grid
- Social Media, Newsletter, Contact, Maps
- And many more components

🎨 **Full Design Control**
- Drag & drop interface
- Real-time preview
- Theme editor with custom fonts
- Background animations (Aurora, particles, gradients)
- Scroll animations

📊 **Advanced Features**
- Collections & dynamic data binding
- Custom forms builder
- Blog engine with posts
- Mobile-responsive design
- SEO optimization

⬇️ **Multiple Export Options**
- Download as ZIP
- Blogger theme XML
- Re-import HTML for editing
- Works offline

🚀 **Live Demo**
Open `index.html` in your browser to start building!

## Quick Start

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Drag sections from the left panel onto the canvas
4. Click sections to edit properties in the inspector
5. Preview and export when ready

## Exporting

1. Click **Export** → Downloads website as ZIP
2. Extract and upload to any host (GitHub Pages, Netlify, Vercel, etc.)
3. Or use **Blogger** button to download Blogger XML theme
